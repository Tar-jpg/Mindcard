# 🃏 MindCard

> พื้นที่พักใจ 3 นาที สำหรับวัยรุ่นกรุงเทพฯ อายุ 13–17 ปี
> การ์ดฝึกใจสั้น ๆ อ้างอิง **DBT** + งานวิจัย **EEG** คณะแพทยศาสตร์ จุฬาฯ

ต้นแบบ (Working Prototype) สำหรับ **HacKaTech — PEOPLE Track** สร้างด้วย Next.js + Tailwind + Framer Motion พร้อม deploy ขึ้น Vercel ได้ทันที

---

## ✨ ฟีเจอร์ในต้นแบบ

| ฟีเจอร์ | อธิบาย |
|---|---|
| **เช็คอินอารมณ์** (Mood Check) | “ตอนนี้รู้สึกยังไง?” ภาษาเป็นกลาง ไม่มีคำว่า “เครียด” |
| **คลังการ์ดฝึกใจ DBT** | 6 การ์ด: หายใจ · Grounding · Body Scan · Self-Soothe · Cold Reset |
| **โหมดเล่นทีละสเต็ป** | ภาพหายใจเคลื่อนไหว + ไทม์เมอร์ + ปุ่มพัก/ข้าม |
| **Reflection** | เช็คอินอีกครั้งหลังเล่น ด้วยภาษาที่ validate ความรู้สึก |
| **สรุปอย่างอ่อนโยน** | ไม่มีคะแนน/ไม่มีถูกผิด สะท้อนใจก่อน→หลังแบบ emoji + ให้กำลังใจ |
| **บันทึกใจ** | ดูความรู้สึกที่สะสม โดยไม่มี streak และไม่มีคะแนนกดดัน |
| **Safety Support** | ปุ่มขอความช่วยเหลือ + สายเด็ก 1387 / 1323 ทุกหน้า + ช่วยเหลือเมื่อยังหนักอยู่ |
| **เสียง & ธีมสบายตา** | เสียงบรรยากาศ (Web Audio) + โทนสีอ่อนโยน |

## 🛡️ ออกแบบให้ปลอดภัยกับวัยรุ่นกลุ่มเปราะบาง

ผู้ใช้คือวัยรุ่น 13–17 ที่อาจมีภาวะซึมเศร้า/คิดทำร้ายตัวเอง การออกแบบจึงอิงหลักฐาน:

- **Validation แทนคำชม** (DBT) — เลี่ยง “เก่งมาก!” ที่อาจทำให้รู้สึกถูกตัดสิน ใช้การยอมรับความรู้สึก
- **ไม่มี streak / คะแนน** — งานวิจัย JMIR พบว่า gamification ไม่ช่วยภาวะซึมเศร้า และ “streak guilt” ยิ่งซ้ำเติมคนที่ perfectionist/ซึมเศร้า
- **เลี่ยง toxic positivity** — ยอมรับว่าบางวันใจไม่ดีขึ้นก็ไม่เป็นไร พร้อมสื่อความหวัง
- **#chatsafe (Orygen)** — ภาษาปลอดภัย ไม่ตีตรา หาความช่วยเหลือได้ง่าย และนำเสนอสายด่วนเมื่อผู้ใช้ยังรู้สึกหนัก
- **สายด่วนที่ตรวจสอบแล้ว** — สายเด็ก 1387 (ต่ำกว่า 18 ปี, ฟรี 24 ชม.) และ 1323 (กรมสุขภาพจิต)

## 🧭 UX แต่ละขั้นตอน

```
หน้าแรก → เช็คอินใจ → เลือกการ์ด → เล่นทีละสเต็ป → ทบทวน → สรุปผล → แดชบอร์ด
  /        /check-in     /cards      /play/[id]     /reflect   /result   /dashboard
```

หน้า `/about` = เบื้องหลังงานวิจัย (DBT + EEG) สำหรับกรรมการ

## 🛠 Tech Stack

- **Next.js 15** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS** — design system โทนสงบ
- **Framer Motion** — แอนิเมชันลื่นไหล
- **lucide-react** — ไอคอน
- ไม่ต้องมี backend/ฐานข้อมูล — เก็บประวัติใน `localStorage` (เหมาะกับ demo และความเป็นส่วนตัว)

## ▲ Deploy ขึ้น Vercel

**วิธีที่ 1 — ผ่าน Vercel CLI (เร็วสุด):**

```bash
npm i -g vercel
vercel          # preview deploy
vercel --prod   # production
```

**วิธีที่ 2 — ผ่าน GitHub + Vercel Dashboard:**

1. push โค้ดขึ้น GitHub repo
2. ไปที่ [vercel.com/new](https://vercel.com/new) → Import repo
3. Framework: **Next.js** (ตรวจจับอัตโนมัติ) → กด **Deploy**
4. ได้ URL จริงสำหรับเปิดบนมือถือทุกเครื่อง

> ไม่ต้องตั้งค่า environment variable ใด ๆ — พร้อม deploy ทันที
